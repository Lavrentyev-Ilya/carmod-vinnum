# DESIGN.md — Дизайн-система каталога автозапчастей

> Стиль: **тёплый профессиональный**. Строгий, инженерно-надёжный, но с воздухом и аккуратной типографикой. Оранжевый сохраняется как фирменный акцент, но перестаёт быть «кричащим» — становится точечным сигналом действия.
> Объём редизайна: **визуальный слой** (цвет, типографика, отступы, тени, состояния). Структура страниц и flow подбора **не меняются**.

---

## 1. Принципы

1. **Воздух важнее плотности.** Текущая проблема №1 — всё прижато друг к другу. Увеличиваем внутренние отступы карточек и строк, добавляем вертикальный ритм.
2. **Один акцентный цвет.** Оранжевый = только действия и важные акценты (CTA, активный таб, цена-триггер). Не использовать оранжевый для декора и фонов больших зон.
3. **Серый — это структура, а не шум.** Фоновый паттерн из деталей убираем или делаем почти невидимым (см. §3). Контент на чистом фоне.
4. **Типографика делает «дорого».** Один качественный гротеск, чёткая иерархия размеров, табличные цифры для цен и артикулов.
5. **Спокойные нейтрали + смысловые цвета.** Зелёный = «в наличии», красный/серый = «нет», синий — справочная информация.

## 3. Фон страницы

**Убрать** плотный паттерн из автодеталей за контентом — он создаёт визуальный шум и удешевляет вид.

- Вариант А (рекомендуется): однотонный `--bg-100`.
- Вариант B (если клиент держится за «автомобильность»): тот же паттерн, но `opacity: 0.03–0.04`, обесцвеченный (grayscale), очень крупный масштаб. Должен едва угадываться, не конкурировать с текстом.

---

## 4. Типографика

**Шрифт:** `Inter` (или `IBM Plex Sans` для чуть более «инженерного» характера). Оба бесплатны, отлично читаются в плотных таблицах, имеют табличные цифры.

```css
font-family: 'Inter', system-ui, -apple-system, 'Segoe UI', sans-serif;
font-feature-settings: 'tnum' 1; /* табличные цифры для цен и артикулов */
```

| Роль | Размер / высота | Начертание | Цвет |
|---|---|---|---|
| H1 (заголовок страницы) | 22 / 30 | 600 | `--ink-900` |
| H2 (секция, «Analogs & Kits») | 18 / 26 | 600 | `--ink-900` |
| Название товара / бренда | 15 / 22 | 600 | `--ink-900` |
| Артикул | 14 / 20 | 500 | `--accent-500` |
| Body / характеристики | 14 / 21 | 400 | `--ink-700` |
| Подписи (мм, кг, label) | 12 / 16 | 400 | `--ink-500` |
| Цена | 18 / 24 | 700, tnum | `--success-600` |

**Правки к текущему виду:** заголовки сейчас тонкие, курсивные и оранжевые — это выглядит дёшево. Делаем их тёмными, плотными (600), курсив убираем. Оранжевым остаётся только акцентный текст (артикул, активный таб).

## 5. Сетка и отступы

Базовая единица — **4px**. Все отступы кратны ей.

| Токен | Значение |
|---|---|
| `--space-1` | 4px |
| `--space-2` | 8px |
| `--space-3` | 12px |
| `--space-4` | 16px |
| `--space-6` | 24px |
| `--space-8` | 32px |

- Внутренний padding карточки товара: `--space-6` (сейчас ~8px — тесно).
- Отступ между строками списка товаров: минимум `--space-4`.
- Контентный контейнер: max-width `1280px`, боковые поля `--space-8`.
- Радиус скругления: карточки `10px`, кнопки/инпуты `8px`, бейджи `6px`.

---

## 6. Тени

Лёгкие, в одном направлении. Без жёстких серых рамок вокруг всего.

```css
--shadow-card: 0 1px 2px rgba(26,29,33,.06), 0 2px 8px rgba(26,29,33,.04);
--shadow-card-hover: 0 4px 16px rgba(26,29,33,.10);
--shadow-popup: 0 12px 32px rgba(26,29,33,.18);
```

Карточки и строки разделяем тенью + тонким бордером `1px var(--ink-300)`, а не толстыми серыми блоками как сейчас.

## 7. Ключевые компоненты

### 7.1 Карточка товара (список)
- Фон `--bg-000`, бордер `1px --ink-300`, радиус `10px`, padding `--space-6`.
- Hover: `--shadow-card-hover`, бордер светлеет к `--accent-100`.
- Бренд+артикул сверху слева, миниатюра слева, характеристики в две колонки (label `--ink-500` / значение `--ink-700`).
- Блок цены справа выровнен, цена `--success-600` 18/700, под ней вес мелким `--ink-500`.

### 7.2 CTA «Add to Cart»
- Фон `--accent-500`, текст белый 600, padding `10px 16px`, радиус `8px`.
- Hover `--accent-600`. Активное нажатие — без сдвига, лёгкое затемнение.
- Иконка корзины слева, `--space-2` до текста.
- «Best Supplier» — вторичная кнопка: бордер `1px --ink-300`, текст `--ink-700`, фон прозрачный.

### 7.3 Бейдж наличия
- In stock: фон `--success-100`, текст `--success-600`, иконка-галочка, радиус `6px`, padding `2px 8px`.
- Out of stock: фон `--bg-100`, текст `--danger-600`.

### 7.4 Заголовок-плашка страницы
- Сейчас: бледно-оранжевая полоса с тонким курсивом. 
- Стало: фон `--accent-100`, текст `--ink-900` 600 (не курсив), слева оранжевая вертикальная полоса 3px `--accent-500` как акцент. Хлебные крошки под плашкой, `--ink-500`, 12px.

### 7.5 Табы (OE Numbers / Suitable vehicles / Reviews)
- Активный: текст `--accent-500` 600, подчёркивание `2px --accent-500`.
- Неактивный: `--ink-500` 400, без рамок-«пузырей».
- Линия-разделитель под всеми табами `1px --ink-300`.

### 7.6 Левое меню разделов (дерево)
- Убрать `+/−` иконки-квадратики на старом стиле, заменить на аккуратный chevron `--ink-500`.
- Активный раздел: текст `--accent-500` 600 + лёгкий фон `--accent-100`.
- Пункт hover: фон `--bg-100`.
- Межстрочный отступ пунктов `--space-2`.

### 7.7 Фильтры (производители, чекбоксы)
- Кастомные чекбоксы: бордер `1px --ink-300`, чек — `--accent-500`.
- Счётчик справа (²,¹) — `--ink-500` 12px, не верхним индексом, а обычным числом в скобках.

### 7.8 Поиск автомобиля (Brand/Model/Engine — экран 1)
- Степпер 1-2-3 оставить, но кружок-номер сделать `--accent-500` только у активного шага, остальные `--ink-300`.
- Инпуты: фон `--bg-000`, бордер `1px --ink-300`, focus — бордер `--accent-500` + тонкое кольцо `0 0 0 3px var(--accent-100)`.

### 7.9 Промо-баннер (25 YEARS)
- Сейчас аляповатый градиент + 3D-иллюстрация. Упростить: тёмно-синий/графитовый фон, крупная цифра, оранжевый акцент только на одном слове. Меньше элементов.

## 8. Состояния (states)

| Состояние | Правило |
|---|---|
| Hover (карточка) | тень усиливается, бордер теплеет |
| Focus (инпут) | кольцо `0 0 0 3px --accent-100` |
| Disabled | opacity .5, cursor not-allowed |
| Loading | скелетон-плейсхолдеры `--bg-100` с лёгким shimmer |
| Selected (строка/раздел) | фон `--accent-100`, левая полоса `--accent-500` |

## 10. Что НЕ трогаем (важно для клиента)

- Расположение блоков и колонок — остаётся.
- Логику подбора Brand → Model → Engine → Section → Part.
- Три режима отображения списка (строки / сетка / компакт).
- Привычные B2B-паттерны: зелёная цена, наличие, «Best Supplier», вес.
